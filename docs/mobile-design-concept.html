2px top border anchors the panel — no drag handle, no rounded cap, just a hard edge
Breadcrumb titlebar shows context without needing a title. Matches desktop panel header pattern.
Property grid — fixed key column, value fills right. Inline controls: stepper, segment, toggle. No stacked label/field pairs.
AI suggestion compressed to a single row — prefix tag | message | apply link. Always visible, never intrusive.
